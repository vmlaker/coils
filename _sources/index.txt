Coils, a Python library
=======================

Coils is a small Python library of generic codes.
It's written in 100% pure Python.

Version 1.0.6 is the latest version of the library,
updated on Thu Feb 13 09:07:19 PST 2014.

Get Coils now!
--------------

Install the module in the usual way, i.e. using pip:
::
   
   pip install coils

You can also clone the repo to muck with the source 
(*even contribute if you want*) -- 
take a look at :doc:`developers`.
